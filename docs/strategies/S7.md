# Strategy 7 — Post-Pump 1H Darvas Range (Bidirectional)

**Directions:** SHORT (breakdown) + LONG (breakout)  
**Timeframes:** 1D (spike/RSI gates) + 1H (Darvas box entry trigger)  
**Config file:** `config_s7.py`  
**Evaluate function:** `evaluate_s7()` in `strategies/s7.py`  
**Sentiment gate:** BEARISH → SHORT only · BULLISH → LONG only · NEUTRAL → HOLD

---

## 1. Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `S7_ENABLED` | `True` | Master enable/disable |
| `S7_BIG_CANDLE_BODY_PCT` | `0.20` | Min 20% body to qualify as spike candle |
| `S7_BIG_CANDLE_LOOKBACK` | `30` | Search last 30 daily candles for spike |
| `S7_RSI_PEAK_THRESH` | `75` | RSI must have reached this level (overbought) |
| `S7_RSI_PEAK_LOOKBACK` | `10` | Last N daily candles to search for RSI peak |
| `S7_RSI_DIV_MIN_DROP` | `5` | RSI bearish divergence: 2nd push ≥5 pts lower than 1st |
| `S7_RSI_STILL_HOT_THRESH` | `70` | Previous daily candle RSI must still be ≥70 |
| `S7_BOX_CONFIRM_COUNT` | `2` | Candles that must hold without new extremum to lock a box level |
| `S7_ENTRY_BUFFER` | `0.005` | SHORT: trigger = box_low × (1−0.5%) · LONG: trigger = box_top × (1+0.5%) |
| `S7_MAX_ENTRY_BUFFER` | `0.04` | Skip if price is already > 4% past trigger |
| `S7_LEVERAGE` | `10` | Trade leverage |
| `S7_TRADE_SIZE_PCT` | `0.04` | 4% of portfolio as margin (initial uses 50% = 2%) |
| `S7_TRAILING_TRIGGER_PCT` | `0.10` | Partial TP: price ±10% from entry |
| `S7_TRAILING_RANGE_PCT` | `10` | 10% trailing callback on remaining 50% |
| `S7_USE_SWING_TRAIL` | `True` | Enable bot-side daily swing trail after partial TP |
| `S7_SWING_LOOKBACK` | `30` | Daily candles for swing trail search |
| `S7_MIN_SR_CLEARANCE` | `0.15` | Skip if S/R level < 15% from entry |

## 2. When to Enter

### Daily gates (identical for both directions — all must pass):

**Step 1 — Big Spike Candle**
- At least one daily candle in the last 30 has a body ≥ 20% of open price (bullish spike)

**Step 2 — RSI Was Overbought**
- Within the last 10 daily candles, RSI (14-period) peaked above 75

**Step 2b — RSI Still Hot**
- The **previous daily candle** (iloc[-2]) RSI must be ≥ 70
- If RSI has cooled below 70, the setup is considered invalidated

**Step 3 — Optional RSI Bearish Divergence**
- First-half RSI peak > second-half RSI peak by ≥ 5 pts → divergence detected
- Informational/boosting, not a hard gate

### 1H Darvas Box detection (same algorithm for both directions):

S7 waits for a **stair-step Darvas box** to form on today's accumulated 1H candles:

1. `today_h1_slice()` slices closed 1H candles since UTC midnight (excludes forming hour)
2. Minimum 6 closed 1H candles required (checked implicitly by `detect_darvas_box`)
3. **Top box** locks first: walking Darvas algorithm scans forward for a high, then waits `S7_BOX_CONFIRM_COUNT` consecutive candles that do not break it upward
4. **Low box** locks second: same algorithm applied to lows after top-box formation
5. The same locked `(box_top, box_low)` pair feeds both directions; only the trigger side differs

The box is re-detected stateless each scan cycle from today's growing 1H slice. At UTC midnight the slate resets.

### SHORT entry (BEARISH sentiment):

- `entry_trigger = box_low × (1 − S7_ENTRY_BUFFER)` — waits for confirmed 1H candle *close* below box_low
- Skip if price already > 4% below box_low (missed move)
- **S/R gate:** `find_spike_base()` locates the pre-pump support base; skip if base < 15% below entry

### LONG entry (BULLISH sentiment):

- `entry_trigger = box_top × (1 + S7_ENTRY_BUFFER)` — waits for confirmed 1H candle *close* above box_top
- Skip if price already > 4% above box_top (missed move)
- **S/R gate:** `find_nearest_resistance()` locates the nearest daily resistance above entry; skip if < 15% away

### Scale-in (both directions)

- Initial entry at **50% of target size** (`S7_TRADE_SIZE_PCT × 0.5`)
- Scale-in queued **1 hour after entry**
- Scale-in window (SHORT): `box_low × (1 − 4%) ≤ mark ≤ box_low × (1 − 0.5%)`
- Scale-in window (LONG): `box_top × (1 + 0.5%) ≤ mark ≤ box_top × (1 + 4%)`
- Scale-in requires matching sentiment (BEARISH for SHORT, BULLISH for LONG)

## 3. When to Exit

### Stop Loss
- **SHORT:** `SL = entry_price × (1 + 50% / leverage)` = −50% P/L stop at 10x leverage
- **LONG:** `SL = entry_price × (1 − 50% / leverage)` = −50% P/L stop at 10x leverage
- Placed atomically with entry via `presetStopLossPrice` on the market order

### Partial TP (50%)
- **SHORT:** 50% closed when price drops 10% from entry
- **LONG:** 50% closed when price rises 10% from entry
- Placed as `profit_plan` order at entry time

### Trailing Stop on Remaining 50%
- Exchange-side `moving_plan` with 10% callback
- **SHORT:** activated at the same −10% trigger level
- **LONG:** activated at the same +10% trigger level

### After Scale-in
- `profit_plan` + `moving_plan` cancelled and re-placed with new total qty
- New trail trigger = `new_avg_entry × (1 ∓ 10%)`

### Bot-Side Swing Trail (post-partial, if `S7_USE_SWING_TRAIL = True`)

Activates only after partial TP has fired.

**SHORT:** mirrors S4
- Each tick: find nearest daily swing high above current price
- If `swing_high < current_SL`, update SL to `swing_high × (1 + entry_buffer)`
- SL only moves downward (never loosens)

**LONG:** mirrors S2
- Each tick: find nearest daily swing low below current price
- If `swing_low > current_SL`, update SL to `swing_low × (1 − entry_buffer)`
- SL only moves upward (never loosens)

## 6. Active Trade Behavior (per tick)

While an S7 trade is open:

1. **PnL sync**: `unrealised_pnl` updated.
2. **Scale-in check**: if `scale_in_pending=True` and 1 hour elapsed, `_do_scale_in()` called.
3. **Partial TP detection**: if live qty drops below 75% of initial, `S7_PARTIAL` logged.
4. **Swing trail** (if enabled and partial fired): SL stepped toward structure (down for SHORT, up for LONG).
5. **Close detection**: when position disappears, `S7_CLOSE` logged.

---

## 7. Key Differences vs S4

| Aspect | S4 | S7 |
|--------|----|----|
| Entry trigger | Previous day's low (1D structure) | 1H Darvas box edge (intraday) |
| Entry confirmed by | Mark price crossing prev_low × buffer | 1H candle *close* beyond box edge |
| Directions | SHORT only | SHORT (BEARISH) + LONG (BULLISH) |
| Sentiment gate | BEARISH | ≠ NEUTRAL |
| Additional 1H filter | `S4_LOW_LOOKBACK` 1H low filter | Darvas box replaces this entirely |
| Swing trail default | `False` | `True` |

## 8. Pair Scanner Display

- `s7_signal` — current signal (`LONG`, `SHORT`, or `HOLD`)
- `s7_reason` — reason string (spike %, RSI peak, Darvas box levels)
- `s7_box_top` / `s7_box_low` — locked Darvas levels
- `s7_sr_support_pct` — spike base clearance % (SHORT path)
- `s7_sr_resistance_pct` — nearest resistance clearance % (LONG path)

## 9. Trade History on Exit

On close, `S7_CLOSE` appended to `trades.csv`. Pair pause rule applies: 3 losses in one UTC day → pair paused until midnight.
